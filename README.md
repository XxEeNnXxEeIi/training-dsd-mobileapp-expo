# 📱 Training DSD Mobile App (Expo)

## 📖 Overview | เกี่ยวกับโปรเจค  
This project was developed as part of the **Department of Skill Development (DSD) Training Program** for **Mobile Application Development**. The application is a **hybrid mobile app** built using **React Native Expo**, demonstrating core mobile development concepts.  

โปรเจคนี้พัฒนาขึ้นเพื่อเป็นส่วนหนึ่งของการฝึกอบรม **หลักสูตรการพัฒนา Mobile Application บนอุปกรณ์ประยุกต์เคลื่อนที่** ของ **กรมพัฒนาฝีมือแรงงาน (DSD)** โดยเป็นแอปพลิเคชันแบบ **Hybrid** ที่สร้างขึ้นด้วย **React Native Expo** ซึ่งแสดงตัวอย่างแนวคิดหลักในการพัฒนาแอปมือถือ  

---

## 🚀 Features | คุณสมบัติของแอป  
✅ **Hybrid App** – Cross-platform mobile application using React Native  
✅ **UI Components** – Sample implementation of buttons, inputs, modals, and navigation  
✅ **API Integration** – Fetch and display data from external sources  
✅ **Offline Support** – Basic offline capabilities using AsyncStorage  
✅ **Theming** – Light and dark mode support  
✅ **Push Notifications** – Example of handling push notifications (if applicable)  

---

## 🛠️ Tech Stack | เทคโนโลยีที่ใช้  
- ⚛️ **React Native Expo** – Framework for hybrid mobile development  
- 🎨 **React Navigation** – Navigation management  
- 🗄️ **AsyncStorage** – Local storage for offline data handling  
- 🌍 **Axios** – Fetching data from APIs  
- 🔔 **Expo Notifications** – Handling push notifications  
- 🎭 **Styled Components / Tailwind CSS** – UI styling

## 🔧 Getting Started | วิธีติดตั้งและใช้งาน  
### 1️⃣ Clone the Repository  
```bash  
git clone https://github.com/yourusername/training-dsd-mobileapp-expo.git  
cd training-dsd-mobileapp-expo  
```

### 2️⃣ Install Dependencies  
```bash  
npm install  
# or  
yarn install  
```

### 3️⃣ Run the Application  
```bash  
npx expo start  
```
Scan the QR code using Expo Go on your mobile device to test the app.
